Manufacturers Meta-Tags — for Zen Cart v2.1.0 and later

Version 3.0.1

Current Support Thread at Zen Cart Forums: https://www.zen-cart.com/showthread.php/129006-manufacturers-meta-tags(keywords-and-title-and-description)

Purpose

This encapsulated plugin enables your store to assign manufacturer-specific meta-tag title, keyword and/or description text, in a manner similar to the product-specific meta-tag processing that is built into Zen Cart. Once the plugin is installed, you'll see additional fields to be entered when you edit (or create) a manufacturer; those values are set into the like-named page metatag when displaying that manufacturer's product listing on your storefront.

  • Metatags Title
  • Metatags Keywords
  • Metatags Description

Notes:

  1. If you have a previous, non-encapsulated, version of this plugin installed, you'll need to remove the files (and core-file overwrite) associated with that version prior to installing an encapsulated version.
  2. This plugin relies on a notification added post-zc210 release to provide the metatags' update to a manufacturer. See this Zen Cart GitHub commit for details. Without this notification present, the metatags won't be stored in the database!

Installation

Follow these steps to install the plugin:

  1. Unzip this distribution's zip-file; you'll see a zc_plugins directory, and inside it a ManufacturersMetatags directory, and inside that another directory with a version number.
  2. Copy the plugin's ManufacturersMetatags directory to your site's file-system's zc_plugins directory.
  3. Log into your Zen Cart admin and go to Modules :: Plugin Manager. Click on Manufacturers Metatags, and click Install, and confirm by clicking Install.

Uninstalling

To uninstall the plugin:

  1. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager.
  2. Locate the entry for Manufacturers Metatags, click the associated Un-Install button and confirm the removal.

Note: Uninstalling the plugin does not remove the additional database columns added to the manufacturers_info table. To remove those fields, enter the following in your admin's Tools :: Install SQL Patches tool:

ALTER TABLE manufacturers_info DROP COLUMN metatags_title, DROP COLUMN metatags_keywords, DROP COLUMN metatags_description;
