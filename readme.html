Manufacturers Meta-Tags — for Zen Cart v1.5.1 through v1.5.4

Version 2.0.0

Current Support Thread at Zen Cart Forums: http://www.zen-cart.com/showthread.php?129006-manufacturers-meta-tags%28keywords-and-title-and-description%29

What it does

This plugin enables your store to assign manufacturer-specific meta-tag title, keyword and/or description text, in a manner similar to the product-specific meta-tag processing that is built into Zen Cart.

Installation

This plugin has one core-file overwrite and one template-override file; you should always backup your cart's database and files prior to making any changes.

  1. Unzip the plugin's package file, which you've most likely already done if you're reading this!
  2. Rename the /YOUR_ADMIN directory to match your Zen Cart's secret admin-folder name.
  3. Rename the one template-override directory (/includes/modules/YOUR_TEMPLATE) to match your cart's active template folder.
  4. Make a copy of your current /YOUR_ADMIN/manufacturers.php file; if you've made previous changes to that file, you'll need to merge those changes with this plugin's updates.
  5. If you've made previous changes to /includes/modules/YOUR_TEMPLATE/meta_tags.php, you'll need to merge those changes with this plugin's updates.
  6. Copy only the plugin's admin-level files to your store's file system.
    1. /YOUR_ADMIN/manufacturers.php
    2. /YOUR_ADMIN/includes/extra_datafiles/meta_tags_manufacturers.php
    3. /YOUR_ADMIN/includes/functions/extra_functions/functions_metatags_manufacturers.php
    4. /YOUR_ADMIN/includes/languages/english/extra_definitions/manufacturers_metatags.php
    5. /YOUR_ADMIN/includes/languages/spanish/extra_definitions/manufacturers_metatags.php
  7. Log into your Zen Cart's admin-console; this will cause the auto-installer to run, adding the database table: meta_tags_manufacturers_description.
  8. Copy the plugin's store-side files to your store's file system.
    1. /includes/languages/english/extra_definitions/meta_tags_manufacturers.php
    2. /includes/modules/YOUR_TEMPLATE/meta_tags.php
